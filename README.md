# MWeb2Markdown


## What is this?

Very simple tool which is converting to MWeb to Markdown format tool.


## Requirements

- Ruby

## Current features

- simple migrate to MWeb to markdown
 - Change image path.

## How to use?


```Examples
$ web2md source_dir/articles dest_dir
```
